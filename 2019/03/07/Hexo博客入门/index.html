Posted on 2019-03-07 | In git

基础知识

  1. 由于github pages存放的都是静态文件，hexo的作用是将本地markdown文件生成html文件，并将页面提交到github。
  2. 为避免不必要的问题，本文所有的命令均在git bash环境下执行。
