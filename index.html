Hexo博客入门

Veröffentlicht am 2019-03-07 | in git

基础知识

  1. 由于github pages存放的都是静态文件，hexo的作用是将本地markdown文件生成html文件，并将页面提交到github。
  2. 为避免不必要的问题，本文所有的命令均在git bash环境下执行。

    环境安装

  3. 下载安装Node.js，网址：https://nodejs.org ，安装成功后界面如下：

  4. 下载安装Git，网址：https://www.git-scm.com/download/ ，git安装成功后，在任意文件夹中右键单击，在上下文菜单中可以看到有Git GUI Here和Git Bash Here两个菜单，在命令行中可以正确执行git命令。

  5. 注册一个github账号，有的话就不用重新注册了。如本人用户名为：beolee，以下均以此为例。
    3.1 新建一个名为”username.github.io”的仓库，如本人的仓库名为”beolee.github.io”，注意这里一定要是你的用户名，其它名称无效，以后博客的访问网址就是：https://beolee.github.io。
    3.2 修改上述仓库的setting中的GitHub Pages设置，Source设置为master。
